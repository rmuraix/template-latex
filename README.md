# LaTeX Template (template-latex)

devcontainerを使用する日本語の論文・レポート向けLaTeXテンプレート．

## Features / 機能

### CI

Automatically lints and builds LaTeX documents using GitHub Actions on push and pull request.

プッシュおよびプルリクエスト時にGitHub Actionsを使用してLaTeXドキュメントの自動リントおよびビルドを行います．

### textlint

Checks the writing quality of Markdown and LaTeX files using [textlint](https://textlint.github.io/). It applies Japanese technical writing rules.

[textlint](https://textlint.github.io/)を使用してMarkdownおよびLaTeXファイルの文章品質を日本語の技術文書ルールでチェックします．

### devcontainer

Provides a pre-configured development container with LaTeX tools and VS Code extensions. This enables a consistent development environment.

LaTeXツールとVS Code拡張機能を含む開発コンテナを提供し，一貫した開発環境を実現します．
